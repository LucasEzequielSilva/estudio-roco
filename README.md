# Estudio Roco

Sitio web del Estudio Roco — Contadores Públicos y Asesores de Empresas (San Salvador de Jujuy).

- Producción: https://estudioroco.com.ar
- Stack: Next.js 16 · React 19 · TypeScript · Tailwind v4

## Desarrollo

```bash
pnpm install
pnpm dev
```

Abrir http://localhost:3000.

## Build

```bash
pnpm build
pnpm start
```
